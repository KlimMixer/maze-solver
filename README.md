## Installation

```bash
$ npm install
```

## Installation development environment

```bash
$ npm install
$ npm run prepare
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# docker development
$ docker-compose up dev

# docker production mode
$ docker-compose up prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
